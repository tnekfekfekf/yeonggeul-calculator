# yeonggeul-calculator
# 영끌계산기 (Yeonggeul Calculator)

**영끌계산기**는 부동산, 대출, 투자 계획 등을 간편하게 계산할 수 있도록 도와주는 모바일 앱입니다.

---

## 📱 주요 기능
- 대출 상환 시뮬레이션
- 총 이자 비용 계산
- 대출 한도 자동 계산
- 간편한 입력 UI
